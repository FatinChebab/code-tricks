Le positionnement en CSS

position

La propriété position en CSS permet de définir la manière dont un élément est positionné dans le document, influençant son placement par rapport aux autres éléments et au conteneur parent. Elle est essentielle pour contrôler la disposition précise des éléments sur une page.
Cette propriété peut avoir pour valeur:

  • Static: est le positionnement adopté par défaut.Il décrit comment les éléments sont positionnés naturellement dans l'ordre où ils sont définis dans le code HTML.
  • Relative: L’élément est positionné par rapport à sa position normale.Il décrit comment les éléments sont positionnés naturellement dans le flux du document.
  • Absolute: L'élément est retiré du flux normal du document et positionné par rapport à son parent positionné ou au viewport.
  • Fixed: Similaire à absolute, mais l'élément est positionné par rapport au viewport. L'élement reste fixe meme en scrollant.
  • Sticky: Combinaison de relative et fixed, l'élément commence en position relative et devient fixe après un certain seuil de défilement.

NB:

Les propriétés suivantes sont utilisées avec position pour définir précisément où un élément sera placé :

  • top : Distance depuis le bord supérieur de l'élément parent.
  • right : Distance depuis le bord droit de l'élément parent.
  • bottom : Distance depuis le bord inférieur de l'élément parent.
  • left : Distance depuis le bord gauche de l'élément parent.

Ces propriétés sont utilisées uniquement si l'élément a une position autre que static.

Utiliser un parent en position: relative pour contrôler le positionnement absolu de ses enfants.
